# Метод `login()`

## Версия
**0.0.1**

## Описание
Метод `login()` является абстрактным методом, предназначенным для реализации логики входа в веб-систему. Метод должен быть реализован в дочерних классах и содержать специфичную логику аутентификации в конкретной системе.

## Сигнатура
```python
def login(self) -> bool
```

## Возвращаемое значение
- **bool**: True, если вход в систему прошел успешно

## Особенности
- Метод является абстрактным и должен быть реализован в дочерних классах
- Возвращает True в случае успешной аутентификации, иначе False
- Является частью основного цикла работы парсера (login -> navigate_to_target -> extract_data -> logout)
- В базовом классе реализация отсутствует (содержит только заглушку)
## Update 2026-03-07 (реализация MultiStepOzonParser)

В конкретной реализации `MultiStepOzonParser.login()` действует проверка активной сессии браузера.

Актуально:
- При успешной проверке сессии метод возвращает `True` (авторизация пропускается, используется сохраненная сессия).
- При исключении во время проверки сессии метод логирует ошибку и возвращает `False`.

Это нужно учитывать при дебаге, так как сбой проверки сессии теперь корректно сигнализирует о проблеме авторизации/состояния браузера.
