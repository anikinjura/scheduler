# Метод `navigate_to_target()` (OzonReportParser)

## Версия
**0.0.2**

## Описание
Метод `navigate_to_target()` переопределяет базовый метод навигации к целевой странице. Сначала выполняет логику родительского класса (`BaseReportParser.navigate_to_target()`), затем добавляет специфичную логику проверки и установки правильного ПВЗ.

## Сигнатура
```python
def navigate_to_target(self) -> bool
```

## Возвращаемое значение
- `True` — если навигация и проверка ПВЗ прошли успешно
- `False` — если не удалось выполнить навигацию или установить правильный ПВЗ

## Алгоритм работы

1. **Вызов родительского метода**:
   - `super().navigate_to_target()` — выполняет базовую навигацию
   - Вычисляет `target_url` из конфигурации
   - Переходит на страницу через `driver.get(target_url)`

2. **Проверка результата базовой навигации**:
   - Если базовая навигация не удалась — возвращает `False`

3. **Проверка и установка ПВЗ**:
   - Вызывает `ensure_correct_pvz()` для проверки и установки правильного ПВЗ
   - Если ПВЗ не установлен — возвращает `False`

4. **Задержка после навигации**:
   - Добавляет задержку для обновления страницы

## Логирование

Метод ведёт подробное логирование:
- `TRACE` — вход в метод
- `DEBUG` — выполнение базовой навигации, проверка и установка ПВЗ
- `ERROR` — ошибки навигации или установки ПВЗ

## Пример использования

```python
from scheduler_runner.utils.parser.core.ozon_report_parser import OzonReportParser

# Создание парсера
parser = OzonReportParser(config)

# Навигация к целевой странице (с проверкой ПВЗ)
success = parser.navigate_to_target()

if success:
    print("Навигация успешна, ПВЗ установлен")
else:
    print("Ошибка навигации или ПВЗ не установлен")
```

## Типичные проблемы

### Ошибка: "Не удалось выполнить базовую навигацию"

**Причина:** Не удалось перейти на целевую страницу (таймаут, ошибка сети, неверный URL).

**Решение:**
1. Проверить доступность URL
2. Проверить валидность сессии браузера
3. Увеличить таймауты в конфигурации

### Ошибка: "Не удалось установить правильный ПВЗ"

**Причина:** Не удалось установить требуемый ПВЗ в dropdown.

**Решение:**
1. Проверить наличие оверлеев на странице
2. Проверить корректность селекторов dropdown
3. Убедиться, что требуемый ПВЗ доступен в списке

### Страница логина вместо отчёта

**Причина:** Сессия истекла или пользователь вышел.

**Решение:**
1. Проверить `driver.current_url` — если содержит 'login', сессия невалидна
2. Обновить сессию (авторизоваться вручную)
3. Проверить `HEADLESS=False` для стабильной работы с сессией
