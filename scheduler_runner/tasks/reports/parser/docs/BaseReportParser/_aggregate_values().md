# Метод `_aggregate_values()`

## Версия
**0.0.1**

## Описание
Метод `_aggregate_values()` реализует агрегацию списка значений по заданному методу. Метод может суммировать, усреднять, подсчитывать количество или находить максимальное/минимальное значение из списка.

## Сигнатура
```python
def _aggregate_values(self, values, aggregation_method, target_field)
```

## Параметры
- **values** (`list`): Список значений для агрегации
- **aggregation_method** (`str`): Метод агрегации ('sum', 'average', 'count', 'max', 'min')
- **target_field** (`str`): Имя поля, в которое будет помещен результат агрегации

## Возвращаемое значение
Словарь с одним полем, содержащим агрегированное значение

## Изменения в версии 0.0.1
- Метод используется в обработке результатов шагов для агрегации значений
- Результаты агрегации могут быть частью результатов шагов, которые содержат `__STEP_SOURCE_URL__`
- `__STEP_SOURCE_URL__` из всех шагов используется для формирования `__SOURCE_URL__` как общего префикса всех URL с помощью метода `_get_common_url_prefix()`

## Пример использования
```python
values = [10, 20, 15, 30]
result = parser._aggregate_values(values, 'sum', 'total')
# Результат: {'total': 75}

result = parser._aggregate_values(values, 'average', 'avg')
# Результат: {'avg': 18.75}
```

## Особенности реализации
- Метод может использоваться в обработке результатов шагов для агрегации значений
- Результаты агрегации могут быть частью результатов шагов, которые содержат `__STEP_SOURCE_URL__`
- `__STEP_SOURCE_URL__` из всех шагов используется для формирования `__SOURCE_URL__` как общего префикса всех URL с помощью метода `_get_common_url_prefix()`
