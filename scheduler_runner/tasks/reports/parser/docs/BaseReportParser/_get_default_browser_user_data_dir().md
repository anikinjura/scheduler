# Метод `_get_default_browser_user_data_dir()`

## Версия
**0.0.1**

## Описание
Метод `_get_default_browser_user_data_dir()` возвращает путь к директории с пользовательскими данными браузера по умолчанию. Метод используется для настройки браузера с использованием существующего профиля пользователя.

## Сигнатура
```python
def _get_default_browser_user_data_dir(self, username: Optional[str] = None) -> str
```

## Параметры
- **username** (`Optional[str]`, optional): Имя пользователя для формирования пути. Если None, используется текущий пользователь

## Возвращаемое значение
- **str**: Путь к директории с пользовательскими данными браузера

## Изменения в версии 0.0.1
- Метод не изменен, но документация обновлена для соответствия стилю проекта
- Метод используется вспомогательными методами для настройки браузера
- Не связан напрямую с формированием `__SOURCE_URL__`, но может использоваться в других частях системы

## Используемые параметры конфигурации
- **BROWSER_USER_DATA_PATH_TEMPLATE** (`str`): Шаблон пути к данным браузера (по умолчанию 'C:/Users/{username}/AppData/Local/Microsoft/Edge/User Data')

## Пример использования
```python
# Получить путь для текущего пользователя
path = parser._get_default_browser_user_data_dir()

# Получить путь для конкретного пользователя
path = parser._get_default_browser_user_data_dir("specific_user")
```

## Особенности
- Метод используется вспомогательными методами для настройки браузера
- Не связан напрямую с формированием `__SOURCE_URL__`, но может использоваться в других частях системы
- Возвращает путь к пользовательским данным браузера, который может использоваться для настройки профиля
