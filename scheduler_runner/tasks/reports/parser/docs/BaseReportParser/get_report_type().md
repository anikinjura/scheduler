# Метод `get_report_type()`

## Версия
**0.0.1**

## Описание
Метод `get_report_type()` является абстрактным методом, предназначенным для возврата типа отчета. Метод должен быть реализован в дочерних классах и возвращать строковое значение, характеризующее тип формируемого отчета.

## Сигнатура
```python
def get_report_type(self) -> str
```

## Возвращаемое значение
- **str**: Тип отчета (например, 'sales_report', 'inventory_report', 'giveout_report')

## Особенности
- Метод является абстрактным и должен быть реализован в дочерних классах
- Возвращает строку, описывающую тип отчета
- Используется для идентификации типа отчета в логике формирования и сохранения отчетов
- В базовом классе реализация отсутствует (содержит только заглушку)
