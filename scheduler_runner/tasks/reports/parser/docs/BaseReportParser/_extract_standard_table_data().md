# Метод `_extract_standard_table_data()`

## Версия
**0.0.1**

## Описание
Метод `_extract_standard_table_data()` извлекает данные из стандартной HTML таблицы с использованием конфигурации колонок. Является вспомогательным методом для `extract_table_data()` и используется для обработки стандартных таблиц.

## Сигнатура
```python
def _extract_standard_table_data(
    self,
    table_element,
    columns_config: list
) -> list
```

## Параметры
- **table_element**: WebElement таблицы, из которой нужно извлечь данные
- **columns_config** (`list`): Список конфигураций колонок, каждая из которых содержит:
  - **name** (`str`): Имя поля в результирующем словаре
  - **selector** (`str`): Селектор ячейки (XPath)
  - **regex** (`str`, optional): Регулярное выражение для извлечения части текста из ячейки

## Возвращаемое значение
- **list**: Список словарей, где каждый словарь представляет собой строку таблицы с именами полей в качестве ключей

## Изменения в версии 0.0.1
- Метод не изменен, но документация обновлена для соответствия стилю проекта
- Метод используется для извлечения данных из таблиц
- Не связан напрямую с формированием `__SOURCE_URL__`, но может использоваться в других частях системы

## Примеры использования
```python
# Пример конфигурации колонок
columns_config = [
    {
        "name": "carriage_number",
        "selector": ".//td[1]",
        "regex": r"(\d+)"
    },
    {
        "name": "status",
        "selector": ".//td[2]"
    }
]

table_data = parser._extract_standard_table_data(table_element, columns_config)
```

## Особенности реализации
- Метод используется для извлечения данных из таблиц
- Не связан напрямую с формированием `__SOURCE_URL__`, но может использоваться в других частях системы
- Является вспомогательным методом для `extract_table_data()`
