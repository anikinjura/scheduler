# Метод `_replace_placeholders_recursive()`

## Версия
**0.0.1**

## Описание
Метод `_replace_placeholders_recursive()` реализует рекурсивную замену плейсхолдеров в структуре данных на значения из словаря замен. Метод обрабатывает вложенные структуры данных любого уровня вложенности, включая строки, словари и списки, и заменяет все найденные плейсхолдеры в формате `{ключ}` на соответствующие значения из переданного словаря.

## Сигнатура
```python
def _replace_placeholders_recursive(self, data, replacements)
```

## Параметры
- **data**: Данные любого типа (строка, словарь, список и т.д.), в которых нужно заменить плейсхолдеры
- **replacements** (`dict`): Словарь с парами ключ-значение для замены плейсхолдеров

## Возвращаемое значение
Обработанные данные с замененными плейсхолдерами

## Изменения в версии 0.0.1
- Метод используется в `_combine_step_results()` для замены плейсхолдеров в результирующей структуре
- Включает замену `{__SOURCE_URL__}` на общий префикс всех `__STEP_SOURCE_URL__` из результатов шагов
- Обрабатывает структуру результата, включающую `__SOURCE_URL__`, который формируется как общий префикс всех `__STEP_SOURCE_URL__` с помощью метода `_get_common_url_prefix()`

## Особенности реализации
- Обработка строк: Если переданный `data` является строкой, метод ищет в ней плейсхолдеры в формате `{ключ}` и заменяет их на соответствующие значения из `replacements`
- Обработка словарей: Если переданный `data` является словарем, метод рекурсивно обрабатывает каждое значение словаря
- Обработка списков: Если переданный `data` является списком, метод рекурсивно обрабатывает каждый элемент списка
- Обработка других типов данных: Для всех остальных типов данных метод возвращает значение без изменений

## Пример использования
```python
data = {
    "source_url": "{__SOURCE_URL__}",
    "summary": {
        "giveout": "{giveout_count}"
    }
}

replacements = {
    "__SOURCE_URL__": "https://turbo-pvz.ozon.ru",
    "giveout_count": 238
}

result = parser._replace_placeholders_recursive(data, replacements)
# Результат: {
#     "source_url": "https://turbo-pvz.ozon.ru",
#     "summary": {
#         "giveout": 238
#     }
# }
```

## Используемые параметры:
- Метод использует `__SOURCE_URL__` из общих замен, который формируется как общий префикс всех `__STEP_SOURCE_URL__` из результатов шагов с помощью метода `_get_common_url_prefix()`
