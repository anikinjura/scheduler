# Метод `_get_common_url_prefix()`

## Версия
**0.0.1**

## Описание
Метод `_get_common_url_prefix()` находит общий префикс для списка URL-адресов. Метод используется для определения общего домена и пути из нескольких URL, что позволяет формировать общий `__SOURCE_URL__` на основе всех `__STEP_SOURCE_URL__` из результатов шагов мульти-шаговой обработки.

## Сигнатура
```python
def _get_common_url_prefix(self, urls: list) -> str
```

## Параметры
- **urls** (`list`): Список URL-адресов, для которых нужно найти общий префикс

## Возвращаемое значение
- **str**: Общий префикс URL-адресов, содержащий схему и домен (например, 'https://turbo-pvz.ozon.ru')

## Пример использования
```python
urls = [
    'https://turbo-pvz.ozon.ru/reports/giveout?filter=...',
    'https://turbo-pvz.ozon.ru/outbound/carriages-archive?filter=...'
]
common_prefix = parser._get_common_url_prefix(urls)
# Результат: 'https://turbo-pvz.ozon.ru'
```

## Особенности реализации
- Метод посимвольно сравнивает компоненты URL (схему, домен, путь) для нахождения общего префикса
- Корректно обрабатывает доменные имена, избегая усечения домена до второго уровня
- Возвращает полный домен, если все входные URL имеют одинаковый домен
