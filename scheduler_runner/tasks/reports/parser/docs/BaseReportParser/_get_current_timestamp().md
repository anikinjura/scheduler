# Метод `_get_current_timestamp()`

## Версия
**0.0.1**

## Описание
Метод `_get_current_timestamp()` возвращает текущую дату и время в формате строки. Метод используется для добавления временных меток в отчеты и логирование.

## Сигнатура
```python
def _get_current_timestamp(self) -> str
```

## Возвращаемое значение
- **str**: Текущая дата и время в формате 'YYYY-MM-DD HH:MM:SS'

## Изменения в версии 0.0.1
- Метод не изменен, но используется в `get_common_report_info()` для получения временной информации
- Временная информация вместе с `__SOURCE_URL__` (который формируется как общий префикс всех `__STEP_SOURCE_URL__` с помощью `_get_common_url_prefix()`) добавляется в общую информацию отчета

## Пример использования
```python
timestamp = parser._get_current_timestamp()
print(f"Текущее время: {timestamp}")
# Вывод: Текущее время: 2023-12-25 10:30:45
```

## Особенности
- Метод возвращает текущую дату и время в формате, подходящем для логирования и отчетов
- Использует формат '%Y-%m-%d %H:%M:%S' для представления времени
- Является вспомогательным методом для получения временных меток
- Используется в методе `get_common_report_info()` для добавления временной информации в отчет
- Временная информация добавляется в общую информацию отчета вместе с `__SOURCE_URL__`, который формируется как общий префикс всех `__STEP_SOURCE_URL__` с помощью метода `_get_common_url_prefix()`
