# Метод `extract_report_data()`

## Версия
**0.0.1**

## Описание
Метод `extract_report_data()` является абстрактным методом, предназначенным для извлечения специфичных данных отчета. Метод должен быть реализован в дочерних классах и содержать логику извлечения данных, специфичных для конкретного типа отчета.

## Сигнатура
```python
def extract_report_data(self) -> Dict[str, Any]
```

## Возвращаемое значение
- **Dict[str, Any]**: Словарь с данными отчета

## Особенности
- Метод является абстрактным и должен быть реализован в дочерних классах
- Возвращает словарь с данными отчета
- Является частью основного цикла работы парсера отчетов
- В базовом классе реализация отсутствует (содержит только заглушку)
