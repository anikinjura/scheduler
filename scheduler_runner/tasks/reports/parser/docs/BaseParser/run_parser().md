# Метод `run_parser()`

## Версия
**0.0.1**

## Описание
Метод `run_parser()` определяет основную последовательность выполнения операций парсера. Метод реализует общий алгоритм: настройка браузера -> вход в систему -> навигация к целевой странице -> извлечение данных -> выход из системы -> закрытие браузера.

## Сигнатура
```python
def run_parser(self) -> Dict[str, Any]
```

## Возвращаемое значение
- **Dict[str, Any]**: Словарь с извлеченными данными

## Исключения
- **Exception**: Возникает в случае ошибки при выполнении любого этапа парсинга

## Алгоритм работы
1. Настройка браузера с помощью метода `setup_browser()`
2. Вход в систему с помощью метода `login()`
3. Навигация к целевой странице с помощью метода `navigate_to_target()`
4. Извлечение данных с помощью метода `extract_data()`
5. Выход из системы с помощью метода `logout()`
6. Закрытие браузера с помощью метода `close_browser()` в блоке finally

## Особенности
- Метод реализует общий алгоритм работы парсера
- В случае ошибки на любом этапе выбрасывается исключение
- Закрытие браузера происходит всегда, даже при возникновении исключения
- В случае ошибки при выходе из системы записывается предупреждение в лог
- Является частью основного интерфейса класса BaseParser
