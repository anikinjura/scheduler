# Метод `extract_data()`

## Версия
**0.0.1**

## Описание
Метод `extract_data()` является абстрактным методом, предназначенным для извлечения данных из веб-системы. Метод должен быть реализован в дочерних классах и содержать логику извлечения специфичных данных из системы.

## Сигнатура
```python
def extract_data(self) -> Dict[str, Any]
```

## Возвращаемое значение
- **Dict[str, Any]**: Словарь с извлеченными данными

## Особенности
- Метод является абстрактным и должен быть реализован в дочерних классах
- Возвращает словарь с извлеченными данными
- В базовом классе реализация отсутствует (содержит только заглушку)
- Предполагается, что в дочерних классах будет реализована специфичная логика извлечения данных
- Является частью основного цикла работы парсера (login -> navigate_to_target -> extract_data -> logout)
