# Метод `logout()`

## Версия
**0.0.1**

## Описание
Метод `logout()` является абстрактным методом, предназначенным для реализации логики выхода из веб-системы. Метод должен быть реализован в дочерних классах и содержать специфичную логику завершения сессии в конкретной системе.

## Сигнатура
```python
def logout(self) -> bool
```

## Возвращаемое значение
- **bool**: True, если выход из системы прошел успешно

## Особенности
- Метод является абстрактным и должен быть реализован в дочерних классах
- Возвращает True в случае успешного завершения сессии, иначе False
- Является частью основного цикла работы парсера (login -> navigate_to_target -> extract_data -> logout)
- В базовом классе реализация отсутствует (содержит только заглушку)
