# Метод `_get_current_user()`

## Версия
**0.0.1**

## Описание
Метод `_get_current_user()` возвращает имя текущего пользователя операционной системы. Метод используется для получения имени пользователя при формировании путей к пользовательским данным браузера.

## Сигнатура
```python
def _get_current_user(self) -> str
```

## Возвращаемое значение
- **str**: Имя текущего пользователя операционной системы

## Пример использования
```python
username = parser._get_current_user()
print(f"Текущий пользователь: {username}")
```

## Особенности
- Метод использует os.getlogin() для получения имени текущего пользователя
- Возвращает имя пользователя в формате, подходящем для использования в путях
- Используется вспомогательными методами для формирования путей к пользовательским данным
- Является вспомогательным методом для работы с пользовательскими данными браузера
