# Метод `_set_checkbox_state()`

## Версия
**0.0.1**

## Описание
Метод `_set_checkbox_state()` устанавливает состояние чекбокса (отмечен или снят). Метод проверяет текущее состояние и изменяет его только при необходимости.

## Сигнатура
```python
def _set_checkbox_state(self, element, target_state: bool) -> bool
```

## Параметры
- **element**: WebElement чекбокса, состояние которого нужно изменить
- **target_state** (`bool`): Целевое состояние чекбокса (True для отмеченного, False для снятого)

## Возвращаемое значение
- **bool**: True, если состояние чекбокса успешно установлено

## Пример использования
```python
checkbox_element = parser.driver.find_element(By.ID, "my-checkbox")
success = parser._set_checkbox_state(checkbox_element, True)  # Отметить чекбокс
if success:
    print("Чекбокс успешно отмечен")
```

## Особенности
- Метод проверяет текущее состояние чекбокса перед изменением
- Изменяет состояние только если оно не соответствует целевому
- Использует клик для изменения состояния чекбокса
- Возвращает False в случае ошибки или если состояние уже соответствует целевому
- Является внутренним методом и используется другими методами класса
