# Метод `extract_table_data()`

## Версия
**0.0.1**

## Описание
Метод `extract_table_data()` извлекает данные из таблицы на веб-странице с использованием конфигурации таблицы. Метод поддерживает различные типы таблиц и позволяет настраивать извлечение данных через конфигурацию.

## Сигнатура
```python
def extract_table_data(
    self,
    table_config_key: str = None,
    table_config: dict = None
) -> list
```

## Параметры
- **table_config_key** (`str`, optional): Ключ конфигурации таблицы в основной конфигурации. Если указан, используется конфигурация из `self.config['table_configs'][table_config_key]`
- **table_config** (`dict`, optional): Прямая конфигурация таблицы. Если указана, используется вместо поиска по ключу.

## Возвращаемое значение
- **list**: Список словарей, где каждый словарь представляет собой строку таблицы с именами полей в качестве ключей

## Структура конфигурации таблицы
Конфигурация таблицы должна содержать следующие поля:
- **table_selector** (`str`): Селектор таблицы (XPath)
- **table_type** (`str`): Тип таблицы ('standard' для обычных таблиц)
- **table_columns** (`list`): Список конфигураций колонок, каждая из которых содержит:
  - **name** (`str`): Имя поля в результирующем словаре
  - **selector** (`str`): Селектор ячейки (XPath)
  - **regex** (`str`, optional): Регулярное выражение для извлечения части текста из ячейки

## Примеры использования

### Извлечение таблицы по ключу конфигурации
```python
# Предполагается, что в конфигурации есть таблица с ключом 'carriages_table'
table_data = parser.extract_table_data(table_config_key='carriages_table')
```

### Извлечение таблицы с прямой конфигурацией
```python
table_config = {
    "table_selector": "//table[@id='my-table']",
    "table_type": "standard",
    "table_columns": [
        {
            "name": "name",
            "selector": ".//td[1]",
        },
        {
            "name": "value",
            "selector": ".//td[2]",
            "regex": r'(\d+)'
        }
    ]
}
table_data = parser.extract_table_data(table_config=table_config)
```

## Особенности
- Метод поддерживает извлечение данных из стандартных HTML таблиц
- Для каждой колонки можно указать регулярное выражение для извлечения части текста
- В случае ошибки возвращает пустой список
- Поддерживает различные типы таблиц (в будущем можно расширить для dynamic таблиц)
