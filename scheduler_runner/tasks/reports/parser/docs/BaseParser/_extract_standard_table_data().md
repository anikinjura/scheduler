# Метод `_extract_standard_table_data()`

## Версия
**0.0.1**

## Описание
Метод `_extract_standard_table_data()` извлекает данные из стандартной HTML таблицы с использованием конфигурации колонок. Является вспомогательным методом для `extract_table_data()` и используется для обработки стандартных таблиц.

## Сигнатура
```python
def _extract_standard_table_data(
    self,
    table_element,
    columns_config: list
) -> list
```

## Параметры
- **table_element**: WebElement таблицы, из которой нужно извлечь данные
- **columns_config** (`list`): Список конфигураций колонок, каждая из которых содержит:
  - **name** (`str`): Имя поля в результирующем словаре
  - **selector** (`str`): Селектор ячейки (XPath)
  - **regex** (`str`, optional): Регулярное выражение для извлечения части текста из ячейки

## Возвращаемое значение
- **list**: Список словарей, где каждый словарь представляет собой строку таблицы с именами полей в качестве ключей

## Примеры использования

### Извлечение данных из найденной таблицы
```python
# Предположим, table_element - это найденный WebElement таблицы
columns_config = [
    {
        "name": "carriage_number",
        "selector": ".//td[1]//div[contains(@class, '_carriageNumber_')]",
        "regex": r'(\d+)'
    },
    {
        "name": "direction",
        "selector": ".//td[2]",
    }
]
rows_data = parser._extract_standard_table_data(table_element, columns_config)
```

## Особенности
- Метод обрабатывает каждую строку таблицы, начиная со второй (пропуская заголовки)
- Для каждой колонки извлекает данные с использованием соответствующего селектора
- Если указано регулярное выражение, извлекает только часть текста, соответствующую паттерну
- В случае ошибки возвращает пустой список
- Является внутренним методом и не предназначен для прямого вызова извне
