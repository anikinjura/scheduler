# Метод `_get_checkbox_state()`

## Версия
**0.0.1**

## Описание
Метод `_get_checkbox_state()` определяет состояние чекбокса (отмечен или снят). Метод используется для проверки текущего состояния чекбокса перед его изменением.

## Сигнатура
```python
def _get_checkbox_state(self, element) -> str
```

## Параметры
- **element**: WebElement чекбокса, состояние которого нужно определить

## Возвращаемое значение
- **str**: Состояние чекбокса ('checked', 'unchecked' или 'unknown')

## Пример использования
```python
checkbox_element = parser.driver.find_element(By.ID, "my-checkbox")
state = parser._get_checkbox_state(checkbox_element)
print(f"Состояние чекбокса: {state}")
```

## Особенности
- Метод проверяет атрибуты и свойства элемента для определения состояния
- Возвращает 'checked' если чекбокс отмечен
- Возвращает 'unchecked' если чекбокс снят
- Возвращает 'unknown' если состояние не может быть определено
- Используется вспомогательными методами для работы с чекбоксами
- Является внутренним методом и используется другими методами класса
