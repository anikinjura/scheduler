# Метод `_terminate_browser_processes()`

## Версия
**0.0.1**

## Описание
Метод `_terminate_browser_processes()` завершает процессы браузера, чтобы освободить системные ресурсы. Метод используется для корректного завершения работы с браузером, особенно в случаях, когда стандартное закрытие не освобождает все процессы.

## Сигнатура
```python
def _terminate_browser_processes(self)
```

## Возвращаемое значение
- **None**: Метод не возвращает значения

## Используемые параметры конфигурации
- **BROWSER_EXECUTABLE** (`str`): Имя исполняемого файла браузера для завершения (по умолчанию 'msedge.exe')
- **PROCESS_TERMINATION_SLEEP** (`int`): Время ожидания после завершения процессов (по умолчанию 2 секунды)

## Пример использования
```python
parser._terminate_browser_processes()
```

## Особенности
- Метод использует subprocess.run для выполнения команды завершения процесса
- Поддерживает завершение пользовательских исполняемых файлов браузера
- Добавляет задержку после завершения процессов для стабильности
- Используется в методе `close_browser()` для полного освобождения ресурсов
- Является вспомогательным методом для корректного завершения работы с браузером
