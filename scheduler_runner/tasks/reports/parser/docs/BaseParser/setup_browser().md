# Метод `setup_browser()`

## Версия
**0.0.1**

## Описание
Метод `setup_browser()` настраивает веб-браузер (Edge) с использованием параметров из конфигурации. Особое внимание уделяется использованию существующего профиля пользователя, что позволяет использовать сохраненную сессию, cookies, настройки и авторизацию пользователя.

## Сигнатура
```python
def setup_browser(self, browser_config: Optional[Dict[str, Any]] = None) -> bool
```

## Параметры
- **browser_config** (`Optional[Dict[str, Any]]`, optional): Конфигурация браузера, которая может переопределить параметры из основной конфигурации

## Возвращаемое значение
- **bool**: True, если браузер успешно настроен

## Поддерживаемые параметры конфигурации
- **BROWSER_EXECUTABLE**: Имя исполняемого файла браузера (по умолчанию 'msedge.exe')
- **EDGE_USER_DATA_DIR**: Путь к пользовательским данным Edge
- **HEADLESS**: Режим headless для браузера
- **PAGE_LOAD_DELAY**: Задержка после загрузки страницы (по умолчанию 3 секунды)

## Примеры использования

### Настройка браузера с параметрами по умолчанию
```python
success = parser.setup_browser()
```

### Настройка браузера с пользовательской конфигурацией
```python
browser_config = {
    "EDGE_USER_DATA_DIR": "C:/Users/username/CustomEdgeProfile",
    "HEADLESS": False
}
success = parser.setup_browser(browser_config)
```

## Особенности
- Метод использует параметр --user-data-dir для указания директории с пользовательскими данными
- Это позволяет использовать сохраненную сессию, cookies, настройки и авторизацию пользователя
- Поддерживает запуск в headless режиме
- В случае ошибки возвращает False
- Автоматически добавляет необходимые аргументы для стабильной работы
