# Метод `set_pvz()`

## Версия
**0.0.1**

## Описание
Метод `set_pvz()` устанавливает нужный ПВЗ (пункт выдачи заказов) в выпадающем списке на странице Ozon. Метод используется для обеспечения правильной локации при сборе данных из системы Ozon.

## Сигнатура
```python
def set_pvz(self, target_pvz: str) -> bool
```

## Параметры
- **target_pvz** (`str`): Целевой ПВЗ для установки в выпадающем списке

## Возвращаемое значение
- **bool**: True, если ПВЗ успешно установлен

## Используемые параметры конфигурации
- **selectors.pvz_selectors.dropdown** (`str`): Селектор выпадающего списка ПВЗ
- **selectors.pvz_selectors.option** (`str`): Селектор опций внутри выпадающего списка

## Пример использования
```python
success = parser.set_pvz("ЧЕБОКСАРЫ_144")
if success:
    print("ПВЗ успешно установлен")
else:
    print("Не удалось установить ПВЗ")
```

## Алгоритм работы
1. Получает селекторы для ПВЗ из конфигурации
2. Использует метод `set_element_value()` для установки значения в выпадающем списке
3. Ждет немного времени для применения изменений
4. Возвращает результат операции

## Особенности
- Метод использует селекторы из конфигурации для поиска элементов
- Поддерживает задержку после установки ПВЗ для стабильности
- Возвращает False в случае ошибки или если селекторы не найдены
- Является частью логики проверки и установки правильного ПВЗ
