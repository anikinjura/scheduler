# Метод `get_current_pvz()`

## Версия
**0.0.1**

## Описание
Метод `get_current_pvz()` извлекает текущий выбранный ПВЗ (пункт выдачи заказов) с помощью селектора из конфигурации. Метод используется для проверки текущего состояния выбора ПВЗ на странице.

## Сигнатура
```python
def get_current_pvz(self) -> str
```

## Возвращаемое значение
- **str**: Текущий выбранный ПВЗ или 'Unknown', если не удалось извлечь

## Используемые параметры конфигурации
- **selectors.pvz_selectors.input_class_readonly** (`str`): Селектор для поля ввода ПВЗ с атрибутом class readonly
- **selectors.pvz_selectors.input_readonly** (`str`): Селектор для поля ввода ПВЗ с атрибутом readonly
- **selectors.pvz_selectors.input** (`str`): Общий селектор для поля ввода ПВЗ

## Особенности
- Метод пытается использовать селекторы в порядке приоритета: input_class_readonly -> input_readonly -> input
- В случае ошибки или если селектор не найден, возвращает 'Unknown'
- Используется в логике проверки и установки правильного ПВЗ
- Является частью функциональности проверки ПВЗ в OzonReportParser
