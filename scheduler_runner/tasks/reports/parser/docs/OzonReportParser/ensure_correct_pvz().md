# Метод `ensure_correct_pvz()`

## Версия
**0.0.1**

## Описание
Метод `ensure_correct_pvz()` проверяет, что на странице выбран правильный ПВЗ (пункт выдачи заказов), и при необходимости устанавливает нужный ПВЗ. Метод используется в процессе навигации для обеспечения правильной локации при сборе данных.

## Сигнатура
```python
def ensure_correct_pvz(self) -> bool
```

## Возвращаемое значение
- **bool**: True, если текущий ПВЗ соответствует ожидаемому или был успешно изменен

## Используемые параметры конфигурации
- **additional_params.location_id** (`str`): ID ожидаемого ПВЗ, извлекается из общей конфигурации

## Пример использования
```python
success = parser.ensure_correct_pvz()
if success:
    print("ПВЗ проверен и установлен правильно")
else:
    print("Не удалось обеспечить правильный ПВЗ")
```

## Алгоритм работы
1. Получает ожидаемый ПВЗ из конфигурации (`additional_params.location_id`)
2. Извлекает текущий ПВЗ с помощью метода `get_current_pvz()`
3. Сравнивает текущий и ожидаемый ПВЗ
4. Если они не совпадают, устанавливает нужный ПВЗ с помощью метода `set_pvz()`
5. Возвращает результат проверки/установки

## Особенности
- Метод автоматически извлекает ожидаемый ПВЗ из конфигурации
- Использует другие методы класса для получения и установки ПВЗ
- Возвращает False в случае ошибки при извлечении или установке ПВЗ
- Является частью логики проверки ПВЗ в методе `navigate_to_target()`
