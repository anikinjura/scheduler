# Модуль отчетов (reports)

Модуль для обработки отчетов из маркетплейсов (ОЗОН, Wildberries, Яндекс.Маркет и др.).

## Структура

- `OzonGiveoutReportScript.py` - скрипт формирования отчета по объемам выдач для маркетплейса ОЗОН
- `GoogleSheetsUploadScript.py` - скрипт загрузки данных отчетов в Google-таблицы
- `NotifyScript.py` - скрипт отправки уведомлений
- `config/` - конфигурационные файлы
  - `reports_list.py` - общие списки для отчетов
  - `reports_paths.py` - пути и переменные для отчетов
  - `reports_schedule.py` - расписание всех скриптов отчетов
  - `scripts/` - конфигурации отдельных скриптов

## Скрипты

### OzonGiveoutReportScript.py
Скрипт формирования отчета по объемам выдач для маркетплейса ОЗОН.

### GoogleSheetsUploadScript.py
Скрипт загрузки данных отчетов в Google-таблицы.

### NotifyScript.py
Скрипт отправки уведомлений.

## Конфигурации

Каждый скрипт имеет свою конфигурацию в формате `script_name_config.py` в директории `config/scripts/`.
